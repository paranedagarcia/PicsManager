# PicsManager

Aplicacion Streamlit para analizar, redimensionar y convertir imagenes PNG, JPG y WEBP.

## Requisitos

- `uv`
- Python 3.11 o superior

## Uso rapido

```bash
uv sync
uv run streamlit run app.py
```
