# PicsManager
CRea un proyecto de app basado en python streamlit para analizar imagenes y realizar las siguientes acciones:

- selecciona una carpeta con las imagenes a analizar o permite cargar imagenes una o varias, formatos permitidos: png, jpg, webp, por default elige 'webp'
- escala las imagenes del paso anterior a una valor predeterminado de 960 pixels de alto o un valor a elegir
- el escalamiento de las imagenes debe ser proporcional alto y ancho
- una vez escalado las imagenes se guardan en el formato elegido (png, jpeg, webp)
- define la mejor alternativa de compresion en cada caso
- en caso de jpeg, se elige porcentaje de resolución o perdida (entre 80 y 95%) mediane un select
. en caso de webp, se elige porcentaje de resolución o perdida (entre 80 y 95%) mediante un select
- las imagenes se guaradn en carpeta 'imagenes'
- las imagenes originales se eliminan por defecto
